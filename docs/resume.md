# Resume

Resume content is derived from the portfolio (skills, projects, aspirations, contact).

## Regenerate PDF

From the project root:

```bash
npm run resume:pdf
```

Output: `resume/Nico-Chikuji-Resume.pdf`

## Edit

- **resume/resume.md** — Edit this file to change content; then run `npm run resume:pdf` again.
- **resume/resume-pdf.css** — Optional: tweak PDF styling (fonts, colors, spacing).
